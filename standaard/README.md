# Standaard GroenWEB site

De pagina's in deze folder beschrijven de functionaliteit en structuur van een standaard drupal 7 GroenWEB site.

## Aanmaken berichtn

* bericht
* pagina
* fiche medewerker
* dossier

## Structuur

* beheerbalk bovenaan
* rechterblokken
* hoofdmenu
* tags beheren

## Andere

* contactformulier
* inhoud zoeken
* wysiwyg editor
